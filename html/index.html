¿Qué es ED GRID?

ED GRID es un framework para Responsive Web Design construido en español, con Sass y mucho café.

Con ED GRID puede crear layouts y prototipos responsive con mucha facilidad, con una metodología fácil de aprender.

ED GRID 1.2 viene con Flexbox. Adios a los floats

Esta versión incluye botones y menús desplegables.
